Making simple plots with FITSFigure
===================================

APLpy includes a :class:`aplpy.FITSFigure` class that provides a simple
interface to make plots and provides easy methods for overlaying shapes, beams,
adding colorbars, and overplotting regions. The interface of this class is
intended to be backward-compatible with existing scripts that used APLpy 1.0 and
earlier, but be sure to read the :ref:`backcompat` if you run into any issues.
